# 🧠 TradeMind AI

AI-powered trading and market analytics platform that combines Machine Learning, Deep Learning, Technical Analysis, Forecasting, Sentiment Analysis, Risk Management, Backtesting, Portfolio Intelligence, and Reinforcement Learning to generate smart trading insights.

---

# 🚀 Features

## 📊 Market Analysis

* Real-time market data analysis
* Multi-asset support (Stocks, Crypto, Forex)
* Technical indicator generation
* Candlestick and advanced chart visualization

## 🤖 AI & Machine Learning Models

* ARIMA Forecasting
* Prophet Forecasting
* LSTM Deep Learning Model
* XGBoost Direction Prediction
* GARCH Volatility Forecasting
* Reinforcement Learning Trading Agent

## 📈 Technical Indicators

* RSI
* MACD
* Bollinger Bands
* SMA / EMA
* Stochastic Oscillator
* ATR

* Volatility Indicators

## 🧪 Backtesting Engine

* RSI Strategy
* MACD Crossover
* Bollinger Reversion
* SMA Golden Cross
* Combined Indicator Strategies

## 📰 Sentiment Analysis

* News sentiment analysis
* Market mood detection

* TextBlob-based NLP analysis

## 💼 Portfolio Intelligence

* Portfolio tracking
* Risk analysis
* Portfolio health scoring

* Market crash simulation

## ⚠️ Risk Management

## 🔔 Smart Alerts

* Trading signal alerts

# 📦 Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/TRADEMIND_AI.git
cd TRADEMIND_AI
```

## 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

## 3️⃣ Run Application

```bash
streamlit run app.py
```
